# Zombie_Survival
Zombie_Survival
뱀파이어 서바이벌(뱀서라이크) 스타일의 게임 
:좀비 서바이벌(임의)

*컨셉: 좀비 아포칼립스 
*전체적인 밸런스는 추후 조정
팀구성
팀장: 송정은 (2071356) → 플레이어, 적, 무기, 아이템 담당
팀원: 손미강 (2271101) → UI, 맵, 사운드 담당

스테이지 구성
총 6스테이지
스테이지당 1분 
스테이지 3, 6 보스 출현 (보스 처치시 다음 스테이지로 이동)
총 플레이타임: 최소 6분

게임 구성
플레이어
기본스펙
체력: MAX 10
이동: WASD
공격: 기본무기(권총 or 돌격소총) +레벨업 무기 
공격력: 무기 공격력 총합
사망 조건: 체력==0
* 몹 처치 수 == 경험치 == 점수
레벨 (MAX 10)
1→2 : 10
2→3 : 20
3→4 : 40
4→5 : 70
5→6 : 110
6→7 : 160
7→8 : 220
8→9 : 290
9→10 : 370
적
잡몹 (97%)
총 2종류 (스테이지 1~3 / 스테이지 4~6) → 색으로 구분
체력: - / -
공격력: - / -
특수몹 (3%)
체력: -
공격력: -
특수 아이템 드랍
회복 (10%)
화면 내 모든 적 제거 (20%)
모둔 무기 쿨타임 초기화 (70%)
보스 
총 2종류 (스테이지 3, 6 출현) 
체력: - / -
공격력: - / - 
맵
스테이지 1~3
숲 컨셉 (무한 스크롤 백그라운드)
출현 적: 잡몹 1, 특수몹, 보스 1(스테이지 3)
스테이지 4~6 
도시(거리) 컨셉 (건물로 상하 이동 제한)
출현 적: 잡몹 2, 특수몹, 보스 2(스테이지 6)
아이템 & 무기
무기 (MAX 3레벨)
기본 무기 - 권총 or 돌격소총 (단일 공격)
공격력: -
쿨타임: -
레벨업 시: 데미지 증가, 쿨타임 감소, 탄 수 증가
근거리 무기 - 야구방망이 (전방 원뿔형 범위 공격)
공격력: -
쿨타임: -
레벨업 시: 데미지 증가, 쿨타임 감소
원거리 무기 - 저격총 (관통 공격)
공격력: -
쿨타임: -
레벨업 시: 데미지 증가, 쿨타임 감소
범위 공격 무기 - ?? (플레이어 주변 원형 범위 지속 데미지)
공격력: -
쿨타임: X
레벨업 시: 데미지 증가, 범위 증가
스탯 (무제한 강화)
이동속도 증가
공격 속도 증가
아이템
특수몹 처치시 획득
회복(10%): 전체 체력의 ??% 회복
화면 내 모든 적 제거 (20%)
모든 무기 쿨타임 초기화 (70%)
시스템
무기 슬롯 3개 (기본 무기 1 + 레벨업 무기 2 선택)
레벨업 시 선택지 제공 (무기 2 + 스탯 1)

UI 구성
상단 좌측
체력 (빨강 바)
경험치 (파랑 바)
상단 중앙
현재 스테이지
경과 시간
상단 우측
점수

게임 흐름
제한시간 1분 동안 몬스터 웨이브
경험치 획득 → 레벨업 → 무기/스탯 선택 및 강화
스테이지 3, 6에서 보스 출현
스테이지 6 클리어 시 게임 엔딩
